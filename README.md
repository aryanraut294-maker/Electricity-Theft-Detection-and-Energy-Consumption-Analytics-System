# Electricity-Theft-Detection-and-Energy-Consumption-Analytics-System

Smart electricity analytics system developed using SQL and Power BI to analyze energy consumption, consumer behavior, billing patterns, and suspicious electricity theft activities.

---

##  Description & Purpose

This project focuses on analyzing electricity consumption and consumer-related data to generate meaningful business insights.

The purpose of this project is to:

- Analyze electricity consumption patterns
- Study consumer billing and payment behavior
- Identify suspicious consumers and possible electricity theft activities
- Monitor district-wise and area-wise performance
- Create interactive dashboards for data-driven decision making

The project converts raw electricity data into visual insights using SQL analysis and Power BI dashboards.

---

## Tech Stack

- SQL
- MySQL
- Power BI
- GitHub

---

## Data Source

**Dataset Used:** Maharashtra Electricity Analytics Synthetic Dataset

The dataset contains the following fields:

- Consumer ID
- District
- Area
- Connection Type
- Units Consumed
- Bill Amount
- Meter Reading
- Payment Status
- Suspicious Flag
- Monthly Consumption Information

##  Features & Highlights

### Dashboard 1: Electricity Overview Dashboard

- Total consumer analysis
- District-wise revenue analysis
- Total electricity unit consumption analysis
- Consumer payment status analysis
- Revenue per consumer analysis

### Dashboard 2: Billing & Consumer Analysis Dashboard

- Average bill amount analysis
- Revenue analysis
- Pending and unpaid consumer analysis
- Area-wise billing analysis
- Average meter reading analysis

### Dashboard 3: Electricity Fraud & Consumer Behavior Dashboard

- Suspicious consumer identification
- District-wise suspicious activity analysis
- Connection-type fraud analysis
- Suspicious consumer percentage analysis
- Consumer behavior monitoring

### SQL Analysis

- Performed 25 SQL queries for data analysis
- Used aggregate functions, filtering, grouping, and subqueries
- Generated business insights for electricity consumption and fraud detection

### Additional Features

- Interactive KPI cards
- Slicers for filtering data
- Dynamic charts and graphs
- Interactive tables and visual reports
